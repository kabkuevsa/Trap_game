# Игра «Ловушки» (C# WinForms)

Простая 2D-игра на C# с использованием Windows Forms.  
Игрок управляет квадратом, избегает врагов и ловит их, расставляя ловушки.

##  Правила игры

- Управляйте **синим квадратом** с помощью **стрелок**.
- Нажмите **Пробел**, чтобы поставить **чёрную ловушку**.
- **Красные враги** движутся случайно и отскакивают от стен.
- Если враг касается ловушки — он исчезает, а ваш **счёт увеличивается**.
- Если вы сталкиваетесь с врагом — игра окончена.
- Победа достигается при наборе **20 очков**.
- Количество ловушек ограничено (зависит от размера поля).

##  Управление

- **Стрелки** — движение игрока
- **Пробел** — установка ловушки
- **Кнопка «Играть»** — начать новую игру

##  Технологии

- Язык: **C#**
- Фреймворк: **Windows Forms (.NET)**
- IDE: **Visual Studio**

##  Как запустить

1. Откройте решение `Trap_Game.sln` в Visual Studio.
2. Нажмите **F5** или выберите **Debug → Start Debugging**.
3. Нажмите **«Играть»** и начните ловить врагов!
